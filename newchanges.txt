Базовая структура кода
Программа на PL/SQL состоит из блоков (анонимных или поименованных). Блок может содержать вложенные блоки, называемые иногда подблоками. Блок состоит из трех основных частей:
1)    секция объявлений (необязательная часть);
2)    тело блока;
3)    обработчики исключений (необязательная часть).
[ <<label_name>> ]	- Метка блока
[DECLARE
 
]	- Секция объявлений
BEGIN	- Тело блока
[EXCEPTION
 
]
END [label_name];	- Обработчики исключений
 
Приведем пример программы, выводящей на экран фразу «Hello, World!».
SET serveroutput ON
DECLARE
   hello VARCHAR2(50) := 'Hello, world!';
BEGIN
   DBMS_OUTPUT.enable;
   DBMS_OUTPUT.put_line(hello);
END;
/ 
Типы данных
Язык PL/SQL поддерживает следующие категории типов:
1)    встроенные типы данных, включая коллекции и записи;
2)    объектные типы данных.
Встроенные типы данных
Встроенные типы данных подразделяются на
1)    скалярные;
2)    составные;
3)    ссылочные;
4)    LOB-типы.
